Projektname:
- Backups

WICHTUG:
- Der bot muss auf den gleichen Server wie die Datenbank Gehostet werden.

Start Bot: 
- Erstell ein Ordner namens backups
- npm i mysqldump
- npm i cron
- npm i fs
- node index.js

Verwendung:
- Der Bot macht Automatisch Backups von deiner SQL Datenbank.
